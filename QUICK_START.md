# SpeakWell - Quick Start Guide

## Welcome to SpeakWell! 🎓

Your professional English training website is ready! Here's how to get started.

## Prerequisites

You need to install these to run the website:

1. **Python 3.8+** - Download from https://www.python.org/
2. **Node.js** - Download from https://nodejs.org/

## Option 1: Automatic Setup (Windows)

Simply run the setup script:

```
setup.bat
```

This will automatically:
- Install Python dependencies
- Install Node.js dependencies
- Build the React frontend

Then you can run the website!

## Option 2: Manual Setup

### Step 1: Install Python Dependencies

```bash
pip install -r requirements.txt
```

### Step 2: Install Node.js Dependencies

```bash
cd frontend
npm install
```

### Step 3: Build React Frontend

```bash
npm run build
```

This creates the production-ready React app in `frontend/build/`

### Step 4: Start the Website

```bash
python main.py
```

Open your browser and go to:
**http://localhost:8000**

## For Development (with Hot Reload)

While backend is running:

```bash
cd frontend
npm start
```

This opens the dev server at **http://localhost:3000** with instant page reloading.

## How to Customize

### Change Instructor Information

Edit `main.py` and find this section:

```python
@app.get("/api/instructor")
async def get_instructor():
    return {
        "name": "Rajesh Kumar",
        # ... update these fields
    }
```

### Add/Update Courses

Edit the `/api/services` endpoint in `main.py` to add or modify courses.

### Update Testimonials

Edit the `/api/testimonials` endpoint to add student reviews.

### Replace Images

Put your own images in `frontend/public/images/`:
- `hero.jpg` - Hero section background
- `instructor.jpg` - Trainer profile picture
- `service1.jpg` - `service6.jpg` - Course images
- `testimonial1.jpg` - `testimonial5.jpg` - Student photos

### Change Colors

All colors are in CSS files. Look for:
- `#667eea` - Primary purple-blue
- `#764ba2` - Secondary dark purple
- `#ff6b6b` - Accent red

Change these values in:
- `frontend/src/App.css`
- `frontend/src/styles/HomePage.css`
- `frontend/src/styles/InstructorPage.css`
- etc.

## Deployment

### Ready to Deploy?

1. Build the frontend:
   ```bash
   cd frontend
   npm run build
   ```

2. The backend automatically serves the built frontend

3. Deploy the FastAPI app:
   - **Heroku**: Use Procfile with gunicorn
   - **PythonAnywhere**: Upload files and configure
   - **AWS/Azure/DigitalOcean**: Use the included Dockerfile
   - **Railway**: One-click deployment from GitHub

## Common Issues

### "python command not found"
- Install Python from https://www.python.org/
- Make sure to check "Add Python to PATH" during installation

### "npm command not found"
- Install Node.js from https://nodejs.org/
- Restart your terminal after installation

### Port 8000 already in use
- Run on a different port: `python main.py --port 8001`
- Or kill the process using port 8000

### Images not showing
- Make sure `frontend/public/images/` has all image files
- Images are generated by `generate_images.py`
- You can replace them with your own images
